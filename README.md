# DIY-servo-gimbal</br>
Making DIY servo gimbal with 3-servo motor, MPU6050 GYRO sensor and Powered with Arduino UNO.

- Use Arduino UNO as MCU
- Gyro sensor MPU6050 at top side of GIMBAL
- 3 of Servo motor driving for Roll & Pitch & Yaw.
